Завдання:
        Допрацюйте сценарій. Якщо користувач ввів значення, яке дорівнює 0 або не є числом - сповістіть про помилку.
        Якщо введені значення коректні - порахуйте об'єм циліндру.
        Якщо конвертація строкового значення в цілочисленне пройшла не успішно, то результат конвертації NaN. Для перевірки значення в умові на рівність із NaN
        потрібно використовувати функцію isNaN(value),яка поверне true якщо значення value не є числом. Така перевірка - value == NaN, не спрацює.
