Завдання:
        Напишіть сценарій гри "вгадай число". Суть гри - вгадати за меншн кількість спроб число, яке загадав сценарій.
        Кожен раз, коли гравець намагається вгадати число, сценарій дає підказку - менше чи більше було загадано число.
        Сценарій загадує число тільки один раз на початку гри і протягом гри воно не повинне змінюватися до завершення гри.
        Для рішення задачі використовуйте циклічні і умовні конструкції.

        для генераціх випадкового числа використовуйте наступний код:
        let randomValue = Math.floor(Math.random() * 101); // случайное значение от 0 до 100 будет записано в переменную randomValue
        
        Приклад гри:
        Сценарій: я загадав випадкове значення від 0 до 100
        Користувач: 50
        Сценарій: я загадав значення більше ніж 50
        Користувачь: 75
        Сценарій: я загадав значення менше ніж 75
        Користувач: 62
        Сценарій: я загадав значення менше ніж 62
        Користувач: 56
        Сценарій: Правильно! Загадане значення 56. Ви вгадали з 4-ї спроби.
